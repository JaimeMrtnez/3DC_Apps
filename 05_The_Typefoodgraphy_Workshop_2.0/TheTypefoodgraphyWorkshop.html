Welcome to The Typefoodgraphy Workshop!

With this mini app, you can create your own type design for your 3D food printer in a quick and easy way.
First, type in the text you want. The maximum length is 10 characters. If you need longer text, simply generate more files. Then, try to manipulate it by changing the base font, or the funniest, click on different parts of the canvas to see how it deforms and takes on a new personality. When you're happy with the result you can change the printer settings and save a PNG image and a G-Code file ready to work with your printer.
Have a good time and Bon Appetit!
